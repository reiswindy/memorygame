# memorygame

Memory card matching game made with CrSFML

## Requirements
- [SFML](https://www.sfml-dev.org/) 2.3 or greater

## Installation

Run from root folder:
```
$ shards --production build --release
```

## Usage

Run from the root folder:
```
$ bin/memorygame
```
